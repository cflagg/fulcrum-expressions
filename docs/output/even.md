## EVEN

Returns number rounded up to the nearest even integer.

### Parameters
`value` : Number (required) - Number to round.

### Returns
Number

### Examples
```
// returns 6
EVEN(5.2)
```

```
// returns 530
EVEN(528.2172814)
```
