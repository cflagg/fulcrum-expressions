## ROUNDDOWN

Rounds a number down to a desired number of decimal places.

### Parameters
`value` : Number (required) - The value to round down.

`places` : Number (required) - The number of places to which to round down.

### Returns
Number

### Examples
```
// returns 156.82
ROUNDDOWN(156.826, 2)
```

```
// returns 156
ROUNDDOWN(156.826, 0)
```
