## ACOS

Returns the inverse cosine of a value, in radians.

### Parameters
`value` : Number (required) - The value for which to calculate the inverse cosine. Must be between -1 and 1, inclusive.

### Returns
Number

### Examples
```
// returns 0.6435011087932843
ACOS(0.8)
```
