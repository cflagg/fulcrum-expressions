## LOG10

Returns the the logarithm of a number, base 10.

### Parameters
`value` : Number (required) - The value for which to calculate the logarithm, base 10.

### Returns
Number

### Examples
```
// returns 2
LOG10(100)
```

```
// returns 1.6989700043360185
LOG10(50)
```
