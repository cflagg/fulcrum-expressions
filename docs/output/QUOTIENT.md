## QUOTIENT

Returns the result of dividing one number by another.

### Parameters
`dividend` : Number (required) - The number to divide.

`divisor` : Number (required) - The number to be divided by.

### Returns
Number

### Examples
```
// returns 6
QUOTIENT(30, 5)
```

```
// returns 0
QUOTIENT(0, 10)
```

```
// returns NaN
QUOTIENT(45, 0)
```
