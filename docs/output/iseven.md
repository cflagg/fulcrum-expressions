## ISEVEN

Checks whether the provided value is an even number.

### Parameters
`value` : Number (required) - The value to be verified as even.

### Returns
Boolean

### Examples
```
// returns true
ISEVEN(4)
```

```
// returns false
ISEVEN(5)
```

```
// returns true
ISEVEN(4.12)
```
