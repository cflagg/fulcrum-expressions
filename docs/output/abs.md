## ABS

Returns the absolute value of a number.

### Parameters
`value` : Number (required) - The number of which to return the absolute value.

### Returns
Number - the absolute value of the `value` parameter 

### Examples
```
// returns 1
ABS(-1)
```

```
// returns 42
ABS(42)
```
