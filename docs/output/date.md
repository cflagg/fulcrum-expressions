## DATE

Converts a provided year, month, and day into a date.

### Parameters
`year` : Number (required) - year

`month` : Number (required) - month

`day` : Number (required) - day

### Returns
Date

### Examples
```
// returns 2015-12-16
DATE(2015, 12, 16)
```
