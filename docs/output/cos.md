## COS

Returns the cosine of a value, in radians.

### Parameters
`value` : Number (required) - The value for which to calculate the cosine. Must be between -1 and 1, inclusive.

### Returns
Number

### Examples
```
// returns 0.15425144988758405
COS(30)
```
